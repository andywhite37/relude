Module OrdExtensions.OrdNamed

OrdNamed contains versions of the comparison functions that have a named argument to disambiguate what's on the lhs vs. rhs of the comparison. The ~compareTo values are on the rhs.

// 10 < 100
let result = 10 |> Int.OrdNamed.lessThan(~compareTo=100);
let lessThan: compareTo:O.t => O.t => bool;

let lte: compareTo:O.t => O.t => bool;
