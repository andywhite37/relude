Module Relude

Relude is an alternative standard library (a "prelude") written in Reason and compiled with the BuckleScript compiler. It builds on the category theory foundation provided by Bastet, and provides a larger collection of types and utility function specialized for those types.

The Relude module serves as a namespace containing all other modules in a convenient hierarchy. This allows you to use Relude.Array without shadowing the Array module provided by the OCaml stdlib.

In many cases, however, you will want to shadow the existing stdlib modules with their Relude equivalents. In this case, you could open Relude; but doing so may shadow more than you want (e.g. the Js module provided by BuckleScript will also be shadowed). See Relude.Globals for a better solution.
